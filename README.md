# NC News Seeding

- Setting Up Environment Variables

Create two environment files:
.env.development
.env.test

Add the following lines to each file:

In .env.development:

PGDATABASE=_name of your DATABASE for DEVELOPMENT_

In .env.test:

PGDATABASE=_name of your DATABASE for TESTING_
